MY PROJECTS

Tip Calculator

This is a handy calculator when going outside with friends and need to quickly calculate the amount of tip that you want to give. It gives you the ability to choose the number of persons, the service quality, it gives you the amount of tip that you've given so far, and the tip history.
This project is my final project from Code4Sp course, the purpose of it is to practice what I've learned so far.

Rock Paper Scissors

In this project I made the logic behind the game, I followed the following step: Understand the Problem, Planing, Pseudo coding, Breaking the problem into smaller problems, Implementing.
The game counts the played rounds and at the fifth round announce the winner. The computer choices are random.
